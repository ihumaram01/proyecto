Conectarse desde Windows

¿Qué necesitas?

Para conectarte a la VPN desde Windows, debes instalar el cliente WireGuard para Windows. Puedes descargarlo desde la página oficial de WireGuard.

Paso a paso

  • Descarga el cliente oficial de WireGuard desde aquí y sigue el proceso de instalación.
  • Abre el cliente de WireGuard y haz clic en el botón "Importar túnel".
  • Escanea el código QR proporcionado o importa el archivo de configuración (.conf).
  • Una vez importado el túnel, haz clic en "Activar" para iniciar la conexión VPN.
  • Ya podrás acceder a los recursos internos del servicio ThinLinc, ya sea mediante el navegador o una aplicación remota.

¿Problemas de conexión?

Si tienes problemas de conexión, verifica lo siguiente:

  • Asegúrate de que la VPN está activada en el cliente WireGuard.
  • Revisa que el archivo de configuración es el correcto.
  • Si persisten los problemas, asegúrate de que tu red no está bloqueando conexiones VPN y que el firewall no está interfiriendo.

Solución de problemas avanzados

Si no logras conectarte, puedes verificar el log de WireGuard para más detalles. Para esto, abre la interfaz de WireGuard, haz clic en "Detalles" y revisa el registro de la conexión.
