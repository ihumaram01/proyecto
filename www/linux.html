Conectarse desde Linux

¿Qué necesitas?

Para conectarte a la VPN desde Linux, necesitarás tener instalado el cliente WireGuard. Este cliente es compatible con la mayoría de las distribuciones de Linux y se puede instalar fácilmente desde los repositorios oficiales.

Paso a paso

  • Abre una terminal y actualiza tus repositorios:
    sudo apt update
  • Instala el paquete de WireGuard:
    sudo apt install wireguard
  • Copia el archivo de configuración que te hemos proporcionado (archivo `.conf`).
  • Mueve el archivo de configuración a la carpeta de WireGuard:
    sudo mv archivo.conf /etc/wireguard/
  • Activa la VPN con el siguiente comando:
    sudo wg-quick up archivo
  • Para verificar la conexión, usa:
    sudo wg

¿Problemas de conexión?

Asegúrate de que el archivo de configuración es correcto y que no haya problemas con la red. Si tienes problemas para conectar, revisa los logs de WireGuard con:

sudo journalctl -u wg-quick@archivo
